Swarming Server

Swarming Server is an open task distribution service written in python running on AppEngine.

  • Usage stats is visible to everyone.
  • Admin pages, well, if you are an admin of this instance.
  • To start a bot, run: 
        mkdir bot; cd bot
    curl -sSLOJ {{host_url}}/get_slave_code
    python swarming_bot.zip
    Don't forget if swarming_bot.zip was already present, curl won't overwrite it so "rm swarming_bot.zip" first if necessary.
  • See the project's page for code and help pages: https://code.google.com/p/swarming/
